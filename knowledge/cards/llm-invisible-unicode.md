# AI/LLM 不可见 Unicode-tag 隐形提示注入

## 适用场景

- 目标含 LLM/Agent，会消费用户或外部文本
- 输入经模型解析且可能含不可见字符
- 存在工具调用/数据外带能力

## 触发信号

- 文本含 Unicode tag / 零宽 / 不可见控制字符
- 过滤按可见字符做，模型按码点解析
- 外部内容被直接拼进 prompt

## 发散问题

- 模型看到的 token 和人看到的文本一致吗？
- 不可见字符能否携带被模型执行的指令？
- 注入能否触发工具调用或数据外带？

## 推荐动作

- 在输入注入不可见 tag 指令，观察模型行为。
- 检查过滤是否按可见字符而漏码点。
- 验证是否改变 agent 动作或泄露上下文。

## 关联 Skills

## 停止条件

- 输入按码点归一化并剥离不可见字符
- 模型不消费不可信文本或无副作用能力

## 检查要求

- 必须证明隐形注入实际改变模型/agent 行为或导致数据外带，且可复现。

## 可晋升经验

- 测 LLM 输入先比"人看到的"与"模型看到的"字节差，不可见字符是隐信道。
